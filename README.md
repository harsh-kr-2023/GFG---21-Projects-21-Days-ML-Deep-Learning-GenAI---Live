# Titanic Dataset - Exploratory Data Analysis

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset
using the ydata-profiling library.

## Objective
To understand the structure, missing values, distributions, correlations,
and potential data quality issues in the dataset.

## Steps Performed
- Loaded raw Titanic dataset
- Generated automated EDA report using ydata-profiling
- Analyzed missing values, outliers, and correlations

## Tools & Libraries
- Python
- Pandas
- ydata-profiling
- Jupyter Notebook

## Output
- `titanic_eda_report.html` (automated EDA report)
- `titanic_eda_report_detail.html` (automated EDA report)
